# КР4 Манжиков Дольган ЭФБО-08-24 Вариант 36
Добавление фильма
Заполните форму в левой части экрана:

Название фильма (обязательно)
 Жанр (опционально)
 Год выпуска (опционально)
 Рейтинг (от 1 до 5 звезд)
 Описание (опционально)

 Нажмите кнопку "Добавить в список"
 Изменение статуса фильма
 В карточке фильма используйте выпадающий список для изменения статуса:
 "Хочу посмотреть" - фильмы в планах

 "Просмотрено" - уже посмотренные фильмы

 Просмотр статистики
 В нижней части страницы отображается статистика:
 Общее количество фильмов
 Количество фильмов "Хочу посмотреть"
 Количество "Просмотренных" фильмов
